Ciência de Dados

Descubra o Futuro com o Curso de Ciência de Dados na Universidade do Estado do Pará Estamos orgulhosos de apresentar o novo curso de Ciência de Dados na Universidade do Estado do Pará, um programa inovador projetado para equipar os alunos com as habilidades necessárias para navegar no mundo complexo dos dados.

O QUE É ESTE CURSO?
O curso de Ciência de Dados combina habilidades analíticas e computacionais para transformar dados em insights significativos. Este curso abrange estatística, programação, aprendizado de máquina, visualização de dados e etc.

SISTEMA DE AVALIAÇÃO

A UEPA conta com um sistema de avaliação geral dividido em três avaliações, sendo a média dessas três a nota geral do estudante

Os discentes que obtiverem a soma das notas das 1ª e 2ª avaliações maior ou igual a 8,0 (oito) estarão aprovados sem necessidade de realizar a avaliação final. Os que obtiverem a soma das notas das 1ª e 2ª avaliações entre 4,0 (quatro) e 7,5 (sete e meio) deverão realizar a avaliação final.

NUCLEO DOCENTE

O Núcleo Docente Estruturante tem função consultiva, propositiva e de assessoramento sobre matéria de natureza acadêmica. O NDE integra a gestão acadêmica do curso que é responsável pela concepção do Projeto Pedagógico do Curso e tem a finalidade de acompanhar a implantação, consolidação e atualização do mesmo. É constituído de três professores do quadro efetivo do curso, com mandato de dois anos.
